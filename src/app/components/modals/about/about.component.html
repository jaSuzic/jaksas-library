About this project: close

This is an online application for library management (renting books). First, there is a login page with following user roles:

  1. Librarian
  2. Admin
  3. Super-Admin

A librarian can add, edit and remove members, books and rents. Admin shares the same functionalities as the librarian, but on top of that, he can also add new users (with the role of librarian or another admin) and remove existing users (except Super-Admin, which is a higher role.) Super-Admin is the only user who can add or edit another Super-Admin along with the same responsibilities as Admin. On this application, you may see :

  • A list of all books in the library with options to add a new book and delete or edit an existing one.
  • A list of all library members, with options to add a new one and delete or edit an existing member.
  • An option to create a new rent for an existing member and existing book with the rent date.
  • A list of all book rents made and also a list of only active rents (if the books are still not returned).
  • An option to return the book with a date of choice.

This application contains an admin panel in which a Librarian can change password or image. Admin has the same options with addition to register a new user, see a list of all users and edit them (except for Super - Admin). A currently active user must be excluded from the users list.

If a certain member or book are removed, all rents linked with them will also be removed. This project is available on the GitHub repo through the following link: Frontend,  Backend
